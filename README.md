# Mini Project CMS

Mini Project CMS adalah aplikasi sederhana berbasis **React** yang memiliki fitur **login authentication** dan **product management** menggunakan API dari DummyJSON.

## 🚀 Live Demo

Aplikasi sudah dideploy dan dapat diakses melalui link berikut:

https://riko-firmando.github.io/mini_project_cms/

---

# 🛠 Tech Stack

- React (Vite)
- React Router
- Axios
- Zustand (State Management)
- TailwindCSS
- DummyJSON API

---

# 📦 Installation & Run Locally

Ikuti langkah berikut untuk menjalankan project di local environment.

### 1. Clone Repository

```bash
git clone https://github.com/riko-firmando/mini_project_cms.git
```

### 2. Masuk ke Folder Project

```bash
cd mini_project_cms
```

### 3. Install Dependencies

```bash
npm install
```

### 4. Jalankan Project

```bash
npm run dev
```

Setelah itu buka browser dan akses:

```
http://localhost:5173
```
